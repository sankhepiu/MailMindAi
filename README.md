# MailMind AI

AI-powered Email Assistant built using FastAPI, Next.js, SQLite, and Google Gemini AI.

## Features

* AI Email Generation
* Email Rewriting
* Tone Adjustment
* Subject Line Generation
* Reply Generation
* Email Summarization
* History Tracking

## Tech Stack

Frontend:

* Next.js
* TypeScript
* Tailwind CSS

Backend:

* FastAPI
* SQLAlchemy
* SQLite

AI:

* Google Gemini API

## Installation

### Backend

```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
